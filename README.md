# Golang Introduction Part 2

A brief (1-2 sentence) introduction to the course, its goals, and its target audience.

## Prerequisites

This course assumes you have completed the [Golang Introduction Course](https://github.com/fatawahid15/education-golang-intro) or have equivalent basic Go programming knowledge.

## Course Sections

This course is divided into several sections, each covering a fundamental concept.

| #  | Section                               | Description                                       |
|----|---------------------------------------|---------------------------------------------------|
| 01 | [Arithmetic Operators](./01-arithmetic-operators/)    | An introduction to arithmetic operators in Go.     |
| 02 | [Arrays](./02-arrays/)    | Covers the basics of arrays in Go.     |
| 03 | [Constants](./03-constants/)    | Explains how to declare and use constants in Go.     |
| 04 | [Switch Case](./04-switch-case/)    | Demonstrates the use of `switch` statements in Go.     |
| 05 | [Variables](./05-variables/)    | Covers variable declaration, scope, and naming conventions in Go.     |
| 06 | [Variadic Functions](./06-variadic-functions/)    | Explains how to use variadic functions in Go.     |

## How to Use This Repository

1.  Clone the repository to your local machine.
2.  Navigate to a section's directory (e.g., `cd 01-arithmetic-operators`).
3.  Read the `README.md` file in that directory for an explanation of the concept.
4.  Run the `main.go` file using `go run main.go` to see the code in action.

Happy coding!
